    <h1>IPL Win Probability Predictor</h1>
    <p>This project contains a comprehensive machine learning pipeline designed to predict the win probability of IPL matches using historical data.</p>

    <h2>🚀 Live Application</h2>
    <p>Test the model predictions in real-time:</p>
    <a href="https://iplprediction711.streamlit.app/" class="btn">View Streamlit Application</a>

    <h2>🛠 Features & Methodology</h2>
    <ul>
        <li><strong>Squad-Level Aggregation:</strong> Computes performance metrics (batting/bowling/economy) for specific playing squads.</li>
        <li><strong>Contextual Features:</strong> Incorporates head-to-head records, venue-specific stats, and toss impact.</li>
        <li><strong>Robust Pipeline:</strong> Standardizes team names and venue identifiers across multiple datasets.</li>
    </ul>
